# Shared Core

This directory is the reusable CS and GATE-style preparation layer shared across HPCL, IFFCO, MSTC interview support, and later exams.

## How to Use This Layer

- Study here first when a topic overlaps across multiple exams.
- Keep exam-specific framing, sources, and schedules in the exam folder, not here.
- Update `.brain/Syllabus.md` and `.brain/WeakAreas.md` when actual progress or gaps change.
