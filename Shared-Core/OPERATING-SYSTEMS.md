# Operating Systems

## Why This Matters Now

- OS is core for HPCL and remains one of the already-confirmed weak areas from BARC.

## High-Yield Topics

- Processes, threads, and system calls
- Scheduling algorithms
- Deadlocks and Banker's algorithm
- Concurrency and synchronization
- Memory management: paging, segmentation, virtual memory
- File systems and disk basics

## Common Traps

- Wrong waiting-time / turnaround-time math
- Forgetting all four deadlock conditions
- Mixing internal vs external fragmentation
- Confusing page replacement policies and Belady's anomaly
