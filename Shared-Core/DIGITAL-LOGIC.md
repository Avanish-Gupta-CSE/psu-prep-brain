# Digital Logic

## Why This Matters Now

- HPCL explicitly includes Digital Logic in the official IS syllabus.
- This is a low-depth but easy-to-drop marks area if revision is delayed.

## High-Yield Topics

- Number systems and conversions
- Boolean algebra identities
- K-maps
- Combinational circuits: adders, subtractors, MUX, DEMUX, encoders, decoders
- Sequential circuits: flip-flops, counters, registers
- Fixed-point vs floating-point representation

## Common Traps

- Mixing up minterms and maxterms
- Forgetting don't-care use in K-map reduction
- Confusing JK, D, and T flip-flop behavior
- Missing overflow/sign interpretation in binary arithmetic
