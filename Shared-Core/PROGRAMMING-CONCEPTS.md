# Programming Concepts

## Why This Matters Now

- HPCL tests broad programming awareness, not only pure CS fundamentals.
- IFFCO role fit depends heavily on how well current resume skills are translated into their stack.

## HPCL Focus

- Java / J2EE fundamentals
- Python basics
- OOP principles
- HTML, CSS, XML, JavaScript, PHP awareness
- Unix / Linux shell programming
- SQL and PL/SQL awareness
- Application server concepts
- React frameworks and REST APIs

## IFFCO Carry-Over

- Node.js, React, REST, and web fundamentals are already strengths
- Oracle SQL / PL/SQL needs explicit work
- C, C#, Entity Framework Core, and app-server framing need gap closure

## Profile Leverage

- Strongest carry-over: React, Node.js, REST APIs, cloud deployment, CI/CD
- Medium carry-over: Java basics from STMicro experience
- Weak carry-over: Oracle, PL/SQL, C#, Entity Framework, IIS/Tomcat specifics
